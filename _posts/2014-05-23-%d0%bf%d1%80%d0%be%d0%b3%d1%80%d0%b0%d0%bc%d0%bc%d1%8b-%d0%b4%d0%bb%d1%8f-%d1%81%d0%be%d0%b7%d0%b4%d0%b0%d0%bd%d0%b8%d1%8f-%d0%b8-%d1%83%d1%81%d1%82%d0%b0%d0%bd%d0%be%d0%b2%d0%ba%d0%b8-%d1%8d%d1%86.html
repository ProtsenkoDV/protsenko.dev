Так уж получилось , что Я начал работать с ЭЦП, и для работы мне нужны разные программы.
Пакет программ предназначен для формирования запросов на получение сертификата, а так же для их установки. Пакет программ проверен лично мной.


В мою подборку входит:
КриптоПРО 3.6.
Программа для удаления следов КриптоПРО.
Арм генерация ключей (Последняя версия на момент 23.05.2014)
Сертификат доверенного коренного центра сертификации.
Сертификат сайта закупок.
Программа для формирования подписи Ланит.

В итоге с помощью данного пакета я легко формировал и устанавливал ключи для закупок, и базгова.
