В данной статье пойдет речь об установке OpenJFX 11 под IntellIiJ IDEA. Установка предельно простая если следовать всем пунктам в точном соответствии.

  1. Загружаем OpenJDK с страницы : https://jdk.java.net/
  2. Далее загружаем OpenJFX с страницы : https://gluonhq.com/products/javafx/
  3. Скачиваем и сразу устанавливаем Scene Builder: https://gluonhq.com/products/scene-builder/
  4. Распаковываем архивы, для удобства я поместил их на диск C: в папку j11
  5. В папке j11 - две папки одна с OpenJDK другая с OpenJFX
  6. В IntelliJ IDEA указываем путь к OpenJDK
  7. Создаем проект JavaFX
  8. В Project Structures на вкладке Libraries добавляем загруженный OpenJFX
  9. В VM Options указываем --module-path C:\j11\openjfx\lib --add-modules=javafx.controls --add-modules=javafx.fxml
  10. Запускаем созданный проект

Хочу обратить ваше внимание на то что Scene Builder не очень работает на 4к экранах.
