Разрешаем все зависимости для установки 1С Предприятие 8.3

Добрый день! Недавно столкнулся с проблемой при установке 1С Предприятие 8.3. на новой версии Ubuntu т.к. он упорно не хотел принимать deb пакет libpng-12 выдавая ошибку (на ранних версиях ubuntu, минте и т.д. этого не встречается)

Необходимо вручную установить данную библиотеку, а потом установить остальные зависимости

Скачатьlibpng-12

Скачатьостальные зависимости (libicu57, libjavascriptcoregtk,libjpeg, libwebkitgtk)



Краткая инструкция по установке libpng-12:

  1. sudo apt install zlib1g-dev
  2. ./configure
  3. make check
  4. sudo make install
  5. sudo ln -s /usr/local/lib/libpng12.{a,so{,.0,0.59.0}} /usr/lib/x86_64-linux-gnu/.

Инструкция по установке остальных зависимостей

  • Распаковываем dependencies.tar
  • Входим в папку
  • sudo dpkg -i *

После этого можете ставить уже 1С Предприятие 8.3 все зависимости будут удовлетворены.
